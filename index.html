About Mapty

Mapty - a workout webapp curated just for your easibility to track and
log your running and cycling workouts without ever forgetting about them.



designed with dedication and technology used with determination for the best user experience. This project is inspired by Jonas Schmedtmann JavaScript course which I undertook and learned a great deal from. Hence, this app goes out to him :)

Integrating leaflet library for map view was the best decision since it has a wide documentation
and was a great start to learn and explore APIs. Moreoever, to tackle and learn more about Pros and Cons of using APIs, I have used Supabase for user authentication and data storage.

Supabase is an open-source Firebase alternative. It is a service that provides a secure and scalable backend for your applications. It is built on top of PostgreSQL and provides a set of APIs for working with your data.
